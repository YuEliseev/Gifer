# Gifer
-------------------------

This is a studing app.
<br/>
It implements giphy.com API using OpenFeign library.

# available mappings:
### /gif/random/{tag}
It returns one random gif and its URL specified by tag.

### /gif/saerch/{search}
It returns 8* gif previews and links to them specified by searching reqest.
<br/>
\* may be changed in applicatin.yaml.

### /gif/get/{id}
It returns the specific gif by id.
